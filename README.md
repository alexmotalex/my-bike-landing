# My Bike Landing

A modern and responsive landing page for a bicycle shop built with HTML, SCSS, and JavaScript.

## Demo

- [DEMO LINK](https://alexmotalex.github.io/my-bike-landing/)

## Features

- Responsive layout for different screen sizes
- Clean and modern UI
- Sections for products, details, about us, and contact

## Tech Stack

- HTML
- SCSS
- JavaScript
- Gulp

## Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
